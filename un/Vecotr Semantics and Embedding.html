Word2vec

The intuition of skip-gram is:

  1. Treat the target word and a neighboring context word as positive examples.

  2. Randomly sample other words in the lexicon to get negative samples.

  3. Use logistic regression to train a classifier to distinguish those two cases.

  4. Use the regression weights as the embeddings.

given a tuple (t , c) of a target word t paired with a candidate context word c

return the probability that c is a real context word

the probability that word c is not a real context word for t is just 1minus

The intuition of the skip-gram model is to base this probability on similarity: a word is likely to occur near the target if its embedding is similar to the target embedding.

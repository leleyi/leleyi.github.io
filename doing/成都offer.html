先看看大家熟知的大厂在成都都有哪些目前已经入驻：

腾讯成都大厦

位于天府三街的腾讯成都分部，相信玩王者荣耀的都知道，哈哈哈！（彩蛋：天美还在多个皮肤背景标识了天府三街）每晚在楼下等女朋友下班的时候，都会看到腾讯的楼层灯火通明，优秀的人都这么努力了，我个咸鱼可得加把劲了～

主要的项目有：手游、企业微信、腾讯云、腾讯视频、QQ浏览器等项目的研发。

岗位：前端、Java、安卓、ios、测试、数据分析、Windows、Mac、DBA、架构、项目管理、商品运营、PM、数据运营、游戏设计、审核、客服、销售等…

腾讯成都大厦A座

阿里巴巴西部基地

位于天府四街的阿里巴巴成都分部，在成都的分公司一般都是P6级别的社招，校园招聘比较少。还有今年差点上市的支付宝（蚂蚁金服），当时可是羡煞我们软件园的程序猿了～

主要的项目有：阿里云、阿里妈妈、国际事业部、蚂蚁金服等项目的研发。

岗位：前端 Web 服务端、Java、ios、移动服务端、大数据开发等…

阿里巴巴西部基地

百度

位于武侯区上善国际，在成都主要是销售公司，研发很少，也没有单独的研发中心，期待后续百度能有更好的发展，能入驻成都。

主要的项目有：智慧城市等项目的研发。

岗位：智慧城市解决方案专家等…

百度

字节跳动

位于天府四街，目前还在不断发展中，主要负责的是抖音的核心业务，包括但不限于IM即时通讯、同城等等。成都现在整个团队还在成长中，坑位多，发展空间也比较大。（PS：紧挨着阿里，银泰城～）

主要的项目有：抖音、基础架构、People 团队、教育、云平台等项目的研发。

岗位：前端、移动端、基础架构、Pass/IaaS 解决方案、数据分析、算法、测试、数据中台、DBA等…

字节跳动

京东

位于西部智谷D区，京东在成都的分公司技术团队不断的在扩编中，目前已经有 500+ 的团队；京东团队的技术氛围比较不错，对于程序猿来说是个不错的地方。

主要的项目有：零售、物流、数科、京东云等项目的研发。

岗位：Java、测开、前端、移动端、NLP、数据分析、电商产品、智能客服产品、AI 产品等…

西部智谷D区京东研发中心

美团

位于天府一街，主要是闪购业务，前后台产品都招，1-3年经验，较少高级研发和管理岗位的坑。

主要的项目有：美团到家、到店、酒旅、金融、基础研发平台等项目的研发。

岗位：Java、测开、架构师、前端、跑腿运营、社群运营、用户运营、商家运营等…

美团

滴滴

位于锦江区天府国际大厦，成都分部很多，高级、资深岗位坑位很多，薪资都不错，可以考虑的选择。

主要的项目有：内部协作、橙心优选等项目的研发。

岗位：移动端、Java、前端、技术顾问、在线协作、流程管理、数据分析等…

滴滴

网易

位于武侯区石羊场，和百度类似，在成都分部研发岗位较少，主要业务集中在教育、销售、培训。

主要的业务有：教育等项目。

岗位：在线辅导老师、业务培训师、辅导主管等…

网易

OPPO

位于天府四街，成都主要的业务是音乐、os、政企等业务，有少量中级和资深岗位。

主要的项目有：平台技术中心、浏览器、手机软件、影像、中间件、智能终端等项目的研发。

岗位：Java、算法、前端、移动端、编译器、架构、OS 内核及虚拟化、Web 引擎、多媒体驱动等…

OCG写字楼OPPO研发中心

爱奇艺

位于天府软件园E区，业务广泛，成都主流业务手游视频、游戏数值、游戏服务器开发、游戏产品运营、3D 动作设计等等。

主要的项目有：手游、AI 算法等项目的研发。

岗位：移动端、算法、数据开发、运维、H5、PHP、Java等…

爱奇艺

华为

位于郫都区高薪西，享受高新西区政策优惠，岗位丰厚，成都地区有大量云服务开发、通用软件、安全大数据、Ai 平台开发等技术岗位。

主要的项目有：网路产品与解决方案、2012实验室、云、消费者、运营商等项目的研发。

岗位：C++、Java、前端、5G、分布式等…

华为研究所

快手

位于新川创新科技园AI创新中心，岗位较少，只有高级 Java 研发岗位，其余都是审核、客服类型岗位。

主要的业务有：Java 高级、内容审核、客服、电商等。

岗位：Java 高级开发、审核、编辑、客服等…

AI创新中心快手部门

新浪

位于天府软件园C区，成都分部研发岗位较少，其余销售岗位、编辑岗位、策划岗位较多。

主要业务有有：绿洲研发、销售、编辑、策划等业务。

岗位：绿洲研发、销售等…

新浪

携程

位于天府四街，招聘岗位也比较少，只有 Java 岗位研发，其余岗位都是产品、旅游管理岗位。

主要的业务有：携程、去哪儿、产品、旅游管理等。

岗位：Java 开发、搜索研发、监控平台研发、客户经理、线路管理等…

携程信息技术大楼

联想

位于郫都区天高新西区，岗位比较丰富，Java、智慧校园、边缘计算、云管平台、物联网都有空缺。

主要的项目有：智慧校园、运管平台等项目的研发。

岗位：Java、服务器技术、产品数据、监控等…

联想西部产业园区

陌陌

位于天府软件园E区，想必男同胞们都用过这个软件。岗位比较完善，高级推荐、ERP、基础中间件等。

主要的项目有：视频直播等项目的研发。

岗位：Java、CI 系统、高级产品等…

陌陌科技

完美世界

位于天府软件园C区，玩游戏的想必都认识，主要有游戏数据运营、游戏测试、角色原画、视觉设计、客户端开发。（PS：每天回家都可以看到～）

主要的项目有：游戏、3D 动画等项目的研发。

岗位：Java、动画师、测试、UI等…

完美世界

锤子科技

位于成华区龙潭工业园，罗永浩重点发展成都片区。也同时吸引了成都广大高校如：川大、电子科技大学、成都信息工程大学的优秀毕业生前往。

主要的项目有：手机、硬件等项目的研发。

岗位：安卓、Java等…

锤子科技

当乐网

位于天府软件园E区，是中国手机游戏下载网站，2014年，研发中心已全部搬迁到成都，技术氛围不错，适合新人成长。

主要的项目有：游戏服务器、游戏视频、客户端、动画、角色建模等项目的研发。

岗位：Java、客户端开发、特效师、服务端开发等…

当乐网

科大讯飞

位于天府软件园E区，是国内首屈一指的以语音技术为产业的软件企业，在国内智能语音领域已占据 70% 的市场。（PS：原来就在家门楼下～）

主要的项目有：智慧医疗、智慧城市、智能汽车等项目的研发。

岗位：Web 前端开发、Java、智能服务开发、测试等…

天府菁蓉中心科大讯飞办公处

聚美优品

位于天府软件园G区，曾一度红遍网络的陈欧的一句“我是陈欧，我为自己代言”，当初可是火遍大江南北。

主要的项目有：社群运营、广告、直播等项目的研发。

岗位：算法、系统工程师、产品视觉动效、UI 等…
